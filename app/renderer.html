Renderer package¶

Package contents¶

This package contains modules related to rendering and processing speech and text-to-speech (TTS) operations.

Modules:

  • renderer: Provides rendering functionalities.

  • speech: Handles speech processing tasks.

  • tts: Manages text-to-speech conversions.

Renderer module¶

This module provides functionalities for managing and rendering conversations.

Bases: object

Manages the rendering of conversations, handling outbound and inbound messages.

Speech module¶

This module handles speech processing tasks, including scheduling and processing events.

class app.renderer.speech.SpeechProcessor¶

Bases: object

Processes speech events using a queue for scheduling.

run()¶

Runs the speech processor, processing events from the queue.

Schedules an event to be processed after a specified delay.

Parameters:

  • delay (int) – The delay before processing the event.

  • action (callable) – The action to perform for the event.

TTS module¶

This module manages text-to-speech (TTS) conversions using the pyttsx3 library.

Functions:

  • convert_text_to_speech: Converts text to speech and saves it as an audio file.

  • audio_synthesis: Synthesizes audio from text and sends it to Asterisk.

  • send_to_asterisk: Sends the audio file to Asterisk for playback.

class app.renderer.tts.TextToSpeechConverter(tts_timeout: int = 10)¶

Bases: object

Converts text to speech using the pyttsx3 library.

convert_text_to_speech(text: str, folder: str)¶

Converts text to speech, saves as an audio file, and returns the content encoded in base64.

Parameters:

  • text (str) – The text to convert to speech.

  • folder (str) – The folder where the temporary audio file will be saved.

Returns:

The base64 encoded content of the generated audio file.

Return type:

str

Raises:

TextToSpeechTimeoutError – If the audio file creation exceeds the timeout.

Synthesizes audio from text and sends it to Asterisk.
